# 📚 Smart Library System

## 📌 Project Description

The **Smart Library System** is a web-based application developed to manage library operations in an easy and efficient way.

It helps students and admins to:

* Manage books 📖
* Issue and return books 🔄
* Track records 📊
* Calculate fines automatically 💰

This system reduces manual work and makes library management faster and more organized.

---

## 🎯 Objective

The main objective of this project is to:

* Automate the library system
* Maintain proper records of books and users
* Reduce human errors
* Provide quick access to information

### 🛠️ Admin Module

* Add new books
* Update book details

* Manage users

---

## 🔄 Working of the System

1. User logs into the system
2. User searches for a book
3. If the book is available, it can be issued
4. The system stores issue date
5. When the book is returned:

   * Due date is checked
   * Fine is calculated if late

---

## 📁 Project Structure

* **Models** → Database structure
* **Controllers** → Logic of the system
* **Routes** → API endpoints
* **Middleware** → Authentication
* **Server** → Main backend file

---

## 🚀 How to Run

1. Clone the project
2. Install dependencies using:

   ```bash
   npm install
   ```
3. Setup `.env` file
4. Run the server:

   ```bash
   npm start
   ```
5. Test APIs using Postman
